# Prerequisites
* Storage Account
* SPN
    -  Read and write role to the storage account (RBAC or ACL)
* Operating System (Linux)
    -  Hadoop
        +  Core Site
        +  TLS Acceleration (secure connection only)
        +  env variables (Hadoop-env.sh)
 * Spark (installation without Hadoop)
     -  env varibales
